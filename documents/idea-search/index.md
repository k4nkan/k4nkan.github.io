# 💡 idea-search

ハッカソンやブレスト向けの**アイデア検索ツール**。
**Tavily AI** や **言語距離**を活用し、関連アイデアを高速検索。

## 📎 リンク
- [こちら](https://idea-search.vercel.app/)

## 🛠 技術スタック

- フレームワーク: Next.js (TypeScript)
- フロント/UI: React
- API: Next.js API Routes
- スタイリング: バニラ CSS

## 🚀 起動方法

```bash
# インストール
npm install

# 開発サーバー起動
npm run dev
```
